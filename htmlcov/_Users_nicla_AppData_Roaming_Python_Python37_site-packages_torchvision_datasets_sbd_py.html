12class SBDataset(VisionDataset): 

13 """`Semantic Boundaries Dataset <http://home.bharathh.info/pubs/codes/SBD/download.html>`_ 

14 

15 The SBD currently contains annotations from 11355 images taken from the PASCAL VOC 2011 dataset. 

16 

17 .. note :: 

18 

19 Please note that the train and val splits included with this dataset are different from 

20 the splits in the PASCAL VOC dataset. In particular some "train" images might be part of 

21 VOC2012 val. 

22 If you are interested in testing on VOC 2012 val, then use `image_set='train_noval'`, 

23 which excludes all val images. 

24 

25 .. warning:: 

26 

27 This class needs `scipy <https://docs.scipy.org/doc/>`_ to load target files from `.mat` format. 

28 

29 Args: 

30 root (string): Root directory of the Semantic Boundaries Dataset 

31 image_set (string, optional): Select the image_set to use, ``train``, ``val`` or ``train_noval``. 

32 Image set ``train_noval`` excludes VOC 2012 val images. 

33 mode (string, optional): Select target type. Possible values 'boundaries' or 'segmentation'. 

34 In case of 'boundaries', the target is an array of shape `[num_classes, H, W]`, 

35 where `num_classes=20`. 

36 download (bool, optional): If true, downloads the dataset from the internet and 

37 puts it in root directory. If dataset is already downloaded, it is not 

38 downloaded again. 

39 transforms (callable, optional): A function/transform that takes input sample and its target as entry 

40 and returns a transformed version. Input sample is PIL image and target is a numpy array 

41 if `mode='boundaries'` or PIL image if `mode='segmentation'`. 

42 """
